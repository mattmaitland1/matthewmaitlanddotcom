Details

My team and I created a model to predict the popularity of a song on Spotify. First, we leveraged their API to generate random datasets. Then, we used K-Means Clustering to Identify features with relevant affects on our target variable. Finally, we built, trained, and evaluated a Random Forest Regressor to predict popularity with a sub-4% error rate. Due to logistical issues, this project is not currently available.
